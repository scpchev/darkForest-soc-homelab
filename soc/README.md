# DarkForest SOC

The SOC section extends the DarkForest infrastructure
with attack simulations, detections and investigations.

## Objectives

- Generate realistic security events
- Validate detection coverage
- Develop Splunk searches
- Develop Wazuh detections
- Practice incident response

## Attack Lab

| Host | Role | Network |
|--------|--------|--------|
| DC1 | Active Directory | 10.10.6.1 |
| Windows11 | User Workstation | 10.10.6.2 |
| Ubuntu24 | Linux Workstation | 10.10.6.3 |

## Scenarios

- Password Spraying
- SSH Brute Force
- SMB Enumeration
- Lateral Movement
- Kerberoasting
